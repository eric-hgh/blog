1、git 回退指定版本

GIT 回退到指定版本的两种方法（reset/revert）

git 回退指定版本

git 回退到指定版本的两种方法：git reset 和 git revert。

  • git reset --hard 目标版本号：会覆盖目标版本之后的所有版本
  • git revert -n 目标版本号：仅仅撤销目标版本的提交，生成一个新的版本
